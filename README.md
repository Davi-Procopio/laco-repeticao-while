# 📌 Laço de Repetição - While (JavaScript)

---

## 📖 Descrição
Este repositório contém exercícios desenvolvidos em **JavaScript** com foco na utilização da estrutura de repetição `while`, visando o aprimoramento da lógica de programação.

---

## 🎯 Objetivo
- Praticar estruturas de repetição (`while`)
- Desenvolver algoritmos funcionais
- Validar dados de entrada
- Utilizar versionamento com Git e GitHub

---

## 🛠️ Tecnologias Utilizadas
- JavaScript  
- Visual Studio Code  
- Git e GitHub  
- Flowgorithm  

---

## 📂 Estrutura do Projeto
O repositório contém **5 arquivos `.js`**, cada um representando um exercício:

1. Validação de número positivo  
2. Média de notas  
3. Total de compras  
4. Validação de nome e sobrenome  
5. Simulação de saque  

---

## ✅ Resultados Esperados
- Programas funcionando corretamente  
- Uso de estruturas de repetição  
- Versionamento no GitHub  

---

## 📌 Observações
Os exercícios foram testados para evitar laços infinitos e garantir o funcionamento correto.
